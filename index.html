Instructions

Each player's turn consists of two moves:

  1. Place a token.
  2. Rotate any tile.

Then the next player takes his turn. The first player to get five tokens in a row wins! The row can be horizontal, vertical, or diagonal.

If all sockets have been filled without any player getting five in a row the game is a draw. If both players get five in a row as a player turns a board, the game is also a draw.
